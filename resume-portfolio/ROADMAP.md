# 개발자 웹 이력서 개발 로드맵

## 📋 프로젝트 개요

개인 포트폴리오용 반응형 웹 이력서 페이지 개발 프로젝트입니다.
모던한 디자인으로 개발자의 경력, 기술스택, 프로젝트 경험을 효과적으로 전시하는 원페이지 이력서입니다.

## 🎯 주요 기능

### 1단계: 기본 구조 (필수)
- [ ] HTML 기본 레이아웃 구성
  - 헤더/네비게이션
  - 자기소개 섹션
  - 경력사항 섹션
  - 기술스택 섹션
  - 프로젝트 섹션
  - 연락처 섹션
  - 푸터

### 2단계: 스타일링 (필수)
- [ ] Tailwind CSS를 이용한 반응형 디자인
- [ ] 모바일/태블릿/데스크톱 최적화
- [ ] 다크/라이트 테마 지원 (선택사항)
- [ ] 부드러운 애니메이션 효과

### 3단계: 인터랙션 (필수)
- [ ] 스무스 스크롤 네비게이션
- [ ] 섹션별 스크롤 진행도 표시
- [ ] 마우스 호버 이펙트
- [ ] 모달/탭 UI

### 4단계: 추가 기능 (선택)
- [ ] 다국어 지원 (한/영)
- [ ] PDF 이력서 다운로드
- [ ] 연락 폼 기능
- [ ] 프로젝트 필터링

## 🛠️ 기술스택

| 분야 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | Tailwind CSS + CSS3 |
| 인터랙션 | Vanilla JavaScript |
| 아이콘 | Font Awesome / SVG |

## 📁 프로젝트 구조 (예상)

```
resume-portfolio/
├── index.html           # 메인 HTML
├── styles/
│   └── tailwind.css     # Tailwind 커스터마이징
├── js/
│   ├── navigation.js    # 네비게이션 기능
│   ├── scroll.js        # 스크롤 이펙트
│   └── theme.js         # 테마 전환
├── assets/
│   ├── images/          # 프로필, 프로젝트 이미지
│   └── icons/           # SVG 아이콘
├── CLAUDE.md            # 프로젝트 개발 가이드
└── README.md            # 배포 및 사용 설명서
```

## 👤 이력서 샘플 구성

### 자기소개
- 이름: 개발자 이름
- 직무: 웹 개발자 / 풀스택 엔지니어
- 한줄 소개: 사용자 중심의 웹 애플리케이션을 개발하는 개발자

### 경력사항
- 회사명, 직급, 기간, 담당 업무 (3-5개 항목)

### 기술스택
- 프론트엔드: HTML, CSS, JavaScript, React
- 백엔드: Node.js, Express, Python
- 기타: Git, Docker, SQL

### 프로젝트 경험
- 프로젝트명, 설명, 기술스택, 링크 (3-5개)

### 연락처
- 이메일, GitHub, LinkedIn, 전화번호

## 🚀 개발 단계

### Phase 1: 기초 개발 (1-2일)
- 프로젝트 초기화
- HTML 시맨틱 구조 작성
- 기본 Tailwind 설정
- 반응형 레이아웃 구현

### Phase 2: 스타일링 및 인터랙션 (2-3일)
- Tailwind CSS로 전체 스타일링
- JavaScript 이벤트 처리
- 스크롤 애니메이션 구현
- 다크테마 토글 기능

### Phase 3: 콘텐츠 및 최적화 (1-2일)
- 실제 이력서 콘텐츠 작성
- 이미지 최적화
- SEO 메타태그 추가
- 크로스 브라우저 테스트

### Phase 4: 배포 (1일)
- GitHub Pages / Vercel / Netlify 배포
- 도메인 설정 (선택사항)
- 최종 점검
