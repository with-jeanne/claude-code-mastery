# 개발자 웹 이력서 포트폴리오 — 최종 구현 계획

## 📋 프로젝트 개요

개인 포트폴리오용 반응형 웹 이력서 페이지 개발 프로젝트입니다.
모던한 디자인으로 개발자의 경력, 기술스택, 프로젝트 경험을 효과적으로 전시하는 원페이지 이력서이며,
한국어, 영어, 일본어 다국어 지원과 다크/라이트 테마 전환 기능을 포함합니다.

## 🎨 주요 섹션 구성

### 1. 헤더 & 네비게이션

- **네비게이션 메뉴**: 자기소개, 경력, 기술, 프로젝트, 연락처
- **언어 선택**: 드롭다운 (🇰🇷 한국어 / 🇺🇸 English / 🇯🇵 日本語)
- **테마 토글**: 달 아이콘 (라이트 모드) / 태양 아이콘 (다크 모드)
- **모바일 메뉴**: 햄버거 아이콘 토글

- 프로필 아바타 이미지 (ui-avatars.com)
- 이름, 직무
- 한줄 소개
- **국적**: 대한민국
- **사용 가능 언어**: 한국어 (원어민), 영어 (비즈니스 레벨)

### 6. 연락처 섹션
- Email, GitHub, LinkedIn 버튼 (아이콘 포함)
- 클릭 시 해당 링크로 이동

### 7. 푸터
- 소셜 링크 (GitHub, LinkedIn, Twitter)
- 빠른 링크 (자기소개, 경력, 프로젝트)
- 저작권 정보

## 🔧 핵심 JavaScript 기능

### js/theme.js — 다크/라이트 테마
```js
// HTML에 'dark' 클래스 추가/제거
// localStorage에 선호 테마 저장
// 아이콘 토글 (fa-moon ↔ fa-sun)
// 시스템 다크모드 자동 감지 (prefers-color-scheme)
```

### js/i18n.js — 다국어 지원
```js
// 3가지 언어 번역 데이터 (ko, en, ja)
// data-i18n 속성 기반 텍스트 교체
// 언어 선택 시 전체 페이지 번역
// localStorage에 언어 선택 저장
```

### js/navigation.js — 네비게이션
```js
// 스무스 스크롤 (클릭 시 해당 섹션으로)
// IntersectionObserver로 활성 섹션 감지
// 활성 섹션 링크 하이라이트
// 모바일 메뉴 토글
```

### js/scroll.js — 스크롤 이펙트
```js
// 스크롤 진행도 바 (상단 그라데이션)
// 섹션 진입 애니메이션 (fadeInUp)
// 헤더 축소 이펙트 (scrolled 클래스)
```

## 🌐 다국어 지원 상세

### 번역 대상 요소
- 네비게이션 메뉴 항목
- 모든 섹션 제목
- 본문 텍스트 (자기소개, 경력 설명, 프로젝트 설명 등)
- 푸터 정보

### 구현 방식
1. HTML의 `data-i18n="key"` 속성으로 번역 키 지정
2. `js/i18n.js`에 3가지 언어 번역 데이터 저장
3. 드롭다운 선택 시 해당 언어의 텍스트로 교체
4. `localStorage`에 선택 언어 저장 (재방문 시 유지)

## 🎨 다크모드 구현

### 기술
- Tailwind CSS `darkMode: 'class'` 전략
- `<html>` 태그에 `dark` 클래스 추가/제거
- 모든 스타일에 `dark:` 접두사로 다크 색상 정의

### 색상 체계
| 요소 | 라이트 모드 | 다크 모드 |
|------|------------|----------|
| 배경 | `bg-white` | `dark:bg-gray-900` |
| 텍스트 | `text-gray-900` | `dark:text-gray-100` |
| 섹션 배경 | `bg-white/gray-50` | `dark:bg-gray-800/900` |
| 카드 | `bg-white` | `dark:bg-gray-800` |
| 뱃지 | `bg-blue-100` | `dark:bg-blue-900` |

## 📱 반응형 디자인

### 브레이크포인트
- **모바일** (< 768px): 햄버거 메뉴, 단열 레이아웃
- **태블릿** (768px ~ 1024px): 2열 그리드, 고정 메뉴
- **데스크톱** (> 1024px): 3열 그리드, 최대 너비 5xl (64rem)

### 반응형 클래스 사용
```html
<!-- 모바일에서 숨김, 데스크톱에서 표시 -->
<ul class="hidden md:flex">...</ul>

<!-- 텍스트 크기 반응형 -->
<h1 class="text-4xl md:text-6xl">...</h1>

<!-- 패딩 반응형 -->
<section class="py-12 md:py-32">...</section>
```
